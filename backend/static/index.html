What is this?

This is a site where you can upload an image of a mathematical expression and it will give you the LaTeX code that would produce that equation!

About us

Hello! We are 4 Computer Science students in the Software Development Class at Harvey Mudd College and created this web app as our project for the class.

During our time as Computer Science majors, we have had to write up homework for different classes using LaTeX, but we’ve learned that it can be a very time consuming task. Thus, we have created a website where you can upload an image and it will actually give you the LaTeX code, cutting the time spent exponentially!

How to Use

1. To select the image you would like to convert to LaTeX, either click and drag an image into the box that says “Drag your files here or click in this area”, or click inside the box and your File Explorer will pop up, allowing you to upload an image.

2. Once you have selected your image, click the upload button in order for the image to be processed.

3. Once the image has been processed, the LaTeX code will appear in a box to the right.

4. If you would like to see what the code produces, click the “Compile LaTeX” button, and it will show what the compiled LaTeX code will produce.

5. If the output isn't entirely correct, go ahead and change it! You can edit that field. And feel free to click "Re-compile LaTeX" as many times as you want!

6. To be able to come back to this code, you can click the “Save Result” button that will provide a permalink that will allow you to come back to this site when your result.
